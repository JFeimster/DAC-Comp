# DAC Compensation Estimator Widget

Embeddable DAC compensation estimator widget.
Calculates:
- Personal commission (Personal Ladder)
- Override Spread
- 5-Level Pay (L1–L5)  
using plan math with configurable BR% assumptions.

## Live Demo
